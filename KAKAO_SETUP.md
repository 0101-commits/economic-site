# 📲 카카오톡 매일 시황 요약 (10·15·22시) — 설정 가이드

매일 **오전 10시 / 오후 3시 / 오후 10시(KST) 무렵** 에 `data.json` 시황이 **본인 카카오톡("나와의 채팅")** 으로
자동 발송됩니다. 제목은 'M/D(요일) 시황' 공통이고, 시각마다 **지수/환율 차트 이미지 1장**이 함께 옵니다:

| 시각(KST) | 차트 |
|---|---|
| **10시** (아침) | 나스닥 + S&P500 |
| **15시** (오후) | 코스피 + 달러-원 |
| **22시** (마감) | 코스피 + 나스닥 |

서버(GitHub Actions)에서 실행되므로 브라우저를 켜둘 필요가 없습니다.
(차트 라벨은 CI 한글폰트 부재 대비 영문이며, 차트를 끄려면 저장소 변수 `KAKAO_CHARTS=0` 을 두면 됩니다.)

> ⏰ 실제 발송은 매시 **정각이 아닌 17분**(예: 10:17)에 1차로 나갑니다. GitHub Actions 스케줄은 정각에
> 전 세계 작업이 몰려 지연·누락되기 쉬워, 정각을 피하고 **42분에 보강 발송**을 한 번 더 둡니다
> (1차가 정상 발송되면 보강은 자동으로 건너뜁니다 — 하루 한 번만 도착).

> 자동 발송은 GitHub Actions(아래 시크릿 2개 필요)가 담당합니다. PlayMCP 의 "나에게 보내기(MemoChat)"
> 도구는 즉석 파일럿/테스트 발송용이며, 매일 정해진 시각의 무인 자동 발송에는 사용할 수 없습니다
> (MCP 도구는 세션이 떠 있어야만 동작하므로). 따라서 정기 루틴은 GitHub Actions 로 구성합니다.

> 카카오톡은 임의의 상대에게 무료 발송이 불가하지만, **"나에게 보내기"(메모 API)** 는
> 무료이고 앱 심사도 필요 없습니다. 개인 데일리 브리핑에 적합합니다.

## 1회 설정 (약 10분)

### ① 카카오 개발자 앱 만들기
1. <https://developers.kakao.com> 로그인 → **내 애플리케이션 → 애플리케이션 추가하기**
2. 생성된 앱 → **앱 키** 의 **REST API 키** 를 복사 (나중에 `KAKAO_REST_API_KEY` 로 사용)

### ② 카카오 로그인 + 메시지 권한 켜기
1. **제품 설정 → 카카오 로그인 → 활성화 ON**
2. **Redirect URI** 등록 — 임시로 `https://localhost` 입력해도 됩니다(토큰 1회 발급용)
3. **제품 설정 → 카카오 로그인 → 동의항목** 에서 **"카카오톡 메시지 전송(`talk_message`)"** 을 **사용함**으로 설정

### ③ refresh_token 1회 발급
1. 아래 주소를 브라우저 주소창에 붙여넣고 이동 (`REST_API_KEY` 를 ①에서 복사한 값으로 교체):
   ```
   https://kauth.kakao.com/oauth/authorize?client_id=REST_API_KEY&redirect_uri=https://localhost&response_type=code&scope=talk_message
   ```
2. 카카오 로그인/동의 후 `https://localhost/?code=XXXXXXXX...` 로 이동됩니다 → 주소의 **`code=` 뒤 값**을 복사
3. 터미널에서 토큰 교환 (둘 다 본인 값으로 교체):
   ```bash
   curl -X POST "https://kauth.kakao.com/oauth/token" \
     -d "grant_type=authorization_code" \
     -d "client_id=REST_API_KEY" \
     -d "redirect_uri=https://localhost" \
     -d "code=붙여넣은_code"
   ```
4. 응답 JSON 의 **`refresh_token`** 값을 복사 (나중에 `KAKAO_REFRESH_TOKEN` 으로 사용)

### ④ GitHub Secrets 등록
저장소 → **Settings → Secrets and variables → Actions → New repository secret** 에서 2개 추가:

| 이름 | 값 |
|---|---|
| `KAKAO_REST_API_KEY` | ①에서 복사한 REST API 키 |
| `KAKAO_REFRESH_TOKEN` | ③에서 복사한 refresh_token |

## 메시지 개인화 (비개발자용)

받는 카톡 메시지의 **모양·문구·구성**은 코딩 없이 **카카오 개발자 콘솔에서 직접** 바꿀 수 있습니다.
값(숫자/등락률)은 자동으로 채워지고, 여러분은 **틀(템플릿)** 만 꾸미면 됩니다.

> ℹ️ 현재 기본 발송은 **차트 이미지 피드**입니다. 아래 콘솔 커스텀 템플릿은 **차트를 끈 경우(`KAKAO_CHARTS=0`)
> 또는 차트 발송이 실패했을 때의 폴백**으로 쓰입니다. 차트 이미지의 구성(어떤 지수/기간)을 바꾸려면
> `scripts/send_kakao_digest.py` 의 `SLOT_CHARTS` 를 수정하면 됩니다(개발자 영역).

**바꾸는 곳**: [카카오 developers](https://developers.kakao.com) → 내 애플리케이션 → 해당 앱 →
**도구 → 메시지 템플릿 → (해당 템플릿) → 템플릿 상세** → 원하는 부분 수정 → 오른쪽 **저장**.
(설정 후 별도 ID 등록 없이도 기본 텍스트로 발송되지만, 이 콘솔 템플릿으로 보내려면 위 ④처럼
`KAKAO_TEMPLATE_ID` 에 템플릿 ID 를 등록하면 됩니다.)

**자동으로 채워지는 변수**(`${KEY}` 형태로 제목·설명 칸에 넣으면 그 자리에 값이 들어갑니다):

| 변수 | 들어가는 내용(예) |
|---|---|
| `${TITLE}` | `6/9(화) 시황` (날짜·요일 머리말) |
| `${SUMMARY}` | 가장 중요한 1~2줄 한줄요약 |
| `${EQ}` | 증시 — 코스피·코스닥·나스닥·S&P |
| `${INV}` | 투자자 순매매 — 외국인·기관·개인(억원) |
| `${FX}` | 환율 — 달러·엔100 |
| `${COM}` | 원자재 — WTI·금·구리 |
| `${BOND}` | 채권 — 한·미 10년물 금리 |
| `${SENT}` | 시장심리 — 공포탐욕지수·VIX |
| `${TOP}` | 종목 — 급등/급락 대표 |

**코딩 없이 할 수 있는 것** (콘솔에서 클릭·입력 후 저장):
- 제목/설명 문구 바꾸기, 보여줄 항목 **추가·삭제·순서 변경** (예: '채권'을 위로, '종목'은 빼기)
- 안 쓰는 섹션 **ON/OFF**, **버튼 이름**(예: '대시보드 보기')·연결 링크 변경
- 위 표의 변수를 원하는 칸에 `${EQ}` 처럼 넣기 (이미 화면의 '사용자 인자' 패널에 목록이 보입니다)

**개발자 도움이 필요한 것** (이 저장소의 `scripts/send_kakao_digest.py` 수정):
- 변수에 들어가는 **값 자체의 계산 방식** 변경(예: 표시 종목 수, 소수점 자리)
- **새로운 변수 추가**(예: `${CRYPTO}` 같이 표에 없는 새 지표). 새 `${KEY}` 는 코드에서 값을 만들어
  넣어줘야 발송 시 채워집니다(콘솔에만 추가하면 빈칸으로 나갑니다).

> 변수 이름 규칙: 영문 대문자·숫자·언더바(`_`)만 사용. 콘솔의 변수와 코드의 키 철자가 **정확히 일치**해야 합니다.

## 참고 / 문제 해결
- **발송 시각**: 매일 약 10:17 / 15:17 / 22:17 KST(정각 혼잡 회피, 누락 시 42분 보강). best-effort 라 몇 분 더 늦을 수 있습니다.
- **차트 이미지**: 슬롯별 차트를 `matplotlib` 로 1장 만들어 **카카오 이미지 업로드 API** 로 올린 뒤 피드로 첨부합니다(사이트 호스팅·도메인 등록 불필요). matplotlib/requests 미설치나 업로드 실패 시 자동으로 텍스트 발송으로 폴백합니다.
- **데이터 기준**: 발송 시점에 저장소에 커밋된 최신 `data.json`(장중 약 10분 주기 갱신본)을 사용합니다.
- **`refresh_token` 만료**: 매일 사용하면 자동 연장됩니다. 만약 만료(약 2개월 미사용 시)되면
  위 **③** 만 다시 수행해 새 `refresh_token` 으로 `KAKAO_REFRESH_TOKEN` 시크릿을 교체하세요.
  (워크플로 로그에 "새 refresh_token 발급" 경고가 뜨면 그 값으로 교체하면 됩니다.)
- **요약 길이**: 카카오 텍스트 템플릿은 200자 제한이라 핵심 지표만 담고, 상세는 대시보드 링크로 제공합니다.
